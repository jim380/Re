# The Re Protocol
**Re** is a blockchain built with Cosmos SDK leveraging Interchain Security. To learn more, visit [here](http://re.cyphercore.io).

## Get started

Install [Go](https://go.dev/dl/)

## Build and Install

```
$ make install
```

## Launch

```
$ red start
```
